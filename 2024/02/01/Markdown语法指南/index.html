Markdown(MD)文档语法使用指南

Markdown 是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。
Markdown 文档从内容到格式，甚至插图，通过键盘就可以通通搞定。现在大多数网站的编辑器都支持Markdown语法。（比如CSDN、语雀、掘金、钉钉文档、石墨文档等都支持Markdown语法）

MD标题的语法为#号+空格，后面带标题文字，一个#到六个#加空格，分别代表一级标题到六级标题。
